인포그래픽 깃허브 배포 완벽 가이드

제미나이로 만든 콘텐츠를 웹사이트로 변환하고 깃허브에 무료로 올리는 방법

Tutorial

15단계로 끝내는 웹 배포 과정

1 제미나이에서 HTML 생성 및 저장

1~3단계: 콘텐츠 변환

제미나이에 접속하여 도구에서 Canvas를 선택합니다. 준비한 설교문이나 내용을 넣고 "깃허브에 올릴 HTML로 만들어줘"라고 요청하세요.

4~5단계: 로컬 파일 저장

Canvas 화면에서 코드 버튼을 클릭하여 전체 내용을 복사합니다. 메모장을 열어 붙여넣기 한 후 index.html이라는 이름으로 저장하세요.

2 깃허브 저장소 만들기 및 파일 업로드

  • 07. 깃허브(GitHub)에 접속하여 로그인합니다.
  • 08. 새로운 저장소(Repository)를 추가(New)합니다.
  • 09. 메뉴바에서 'Add file'을 클릭하여 작성한 index.html 파일을 올립니다.
  • 10. 파일이 다 올라가면 하단의 'Commit changes' 버튼을 누릅니다.

3 웹사이트 활성화 및 배포

페이지 설정 과정 (11~13단계)

  1. • 저장소 상단의 Settings 메뉴를 클릭합니다.
  2. • 왼쪽 사이드바에서 Pages 버튼을 클릭합니다.
  3. • Build and deployment 섹션에서 Branch를 'main'으로 선택하고 Save를 누릅니다.

최종 배포 (14~15단계)

약 1분 정도 기다린 후 페이지를 새로고침하면 상단에 사이트 주소가 나타납니다. 'Visit site'를 클릭하여 주소를 복사하고 공유하세요!
